# Teen Digital Wellbeing Dashboard

## About This Project

This project explores teen digital wellbeing through data analysis and visualization.

I worked with SQL to analyze the dataset and used Excel to create a dashboard that presents the insights visually.

The goal was to better understand how factors such as social media usage, sleep quality, and addiction levels relate to wellbeing and academic performance among teenagers.

---

## Questions I Explored

* Do male and female teens spend different amounts of time on social media?
* Which social media platforms are most commonly used?
* Does sleep quality relate to mental health risk?
* How are teens distributed across digital wellbeing categories?
* Does addiction level affect academic performance?

---

## Tools Used

* SQL – Data cleaning and analysis
* Excel – Dashboard and visualization
* GitHub – Project documentation

---

## Key Insights

* Male teens recorded slightly higher social media usage.
* Most teens used multiple social media platforms.
* Poor sleep quality showed the highest mental health risk.
* Most teens demonstrated moderate digital wellbeing.
* Academic performance remained relatively stable across addiction levels.
